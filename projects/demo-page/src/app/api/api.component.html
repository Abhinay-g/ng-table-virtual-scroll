Directives

TableItemSizeDirective

A virtual scroll strategy for mat-table that supports fixed-size items.

Selector: cdk-virtual-scroll-viewport[tvsItemSize]

Properties
Name Description
@Input()
headerHeight: number; 		The size of the header row in the table (in pixels). Defaults to 56px.
@Input()
rowHeight: number; 		The size of the row in the table (in pixels). Defaults to 48px.
@Input()
bufferMultiplier: number; 		The amount of buffer rendered beyond the viewport will be equal to the size of the table 'bufferMultiplier' times; Defaults to 1.7.
