Directives

TableItemSizeDirective

A virtual scroll strategy for mat-table that supports fixed-size items.

Selector: cdk-virtual-scroll-viewport[tvsItemSize]

Properties
Name Description
@Input()
tvsItemSize: number; 		The size of the row in the table (in pixels). Defaults to 48px.
@Input()
headerHeight: number; 		The size of the header row in the table (in pixels). Defaults to 56px.
@Input()
bufferMultiplier: number; 		The 'bufferMultiplier * visibleRowsCount' number of rows will be rendered before and after visible part of the table. Defaults to 0.7.
